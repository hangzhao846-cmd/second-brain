# 笔记类型体系

笔记类型不是目录分类，而是知识加工方式。每篇笔记优先判断它要产出什么能力资产。

| 类型 | 用途 | 什么时候创建 | 命名方式 | 示例 |
|---|---|---|---|---|
| 概念笔记 | 定义术语和边界 | 遇到高频概念但理解不稳 | `概念 - 名称` | `概念 - Safepoint` |
| 机制笔记 | 解释系统如何运转 | 需要理解底层机制和失败场景 | `机制 - 对象 - 机制` | `机制 - G1 - Mixed GC` |
| 架构笔记 | 分析系统结构和权衡 | 设计模块、拆解系统、阅读架构 | `架构 - 系统 - 主题` | `架构 - MMORPG - 场景服线程模型` |
| 故障案例 | 沉淀事故和异常 | 线上故障、压测异常、严重 Bug | `故障 - 日期 - 现象` | `故障 - 2026-05-31 - 网关延迟尖刺` |
| 性能分析 | 记录性能问题定位 | 压测、优化、容量评估 | `性能 - 系统 - 指标异常` | `性能 - 战斗服 - P99 抖动` |
| 代码阅读 | 记录源码理解 | 阅读 JDK、Netty、项目核心代码 | `代码阅读 - 项目 - 模块` | `代码阅读 - Netty - EventLoop` |
| ADR | 记录工程决策 | 有重要取舍且未来会被追问 | `ADR-编号 - 决策主题` | `ADR-0001 - 网关连接状态存储策略` |
| 项目复盘 | 复盘项目结果 | 项目结束或阶段结束 | `项目复盘 - 项目 - 阶段` | `项目复盘 - 压测平台 - MVP` |
| 学习复盘 | 把学习转为能力 | 学完书、课程、源码专题 | `学习复盘 - 主题 - 时间` | `学习复盘 - JMM - 2026W22` |
| Prompt 模板 | 沉淀 AI 工作流 | 某类任务可稳定复用 | `Prompt - 任务` | `Prompt - 生成故障排查树` |
| Checklist | 执行检查 | 高频任务需要防漏项 | `Checklist - 场景` | `Checklist - Java 服务上线前检查` |
| 实验记录 | 验证假设 | 做 benchmark、PoC、压测 | `实验 - 假设 - 日期` | `实验 - 虚拟线程对游戏逻辑线程的影响 - 2026-05-31` |
| 论文/文章阅读 | 提取外部输入 | 阅读高价值材料 | `阅读 - 作者或来源 - 标题` | `阅读 - Netty Docs - Thread Model` |
| 工具使用记录 | 记录诊断工具 | 学习或使用工具后 | `工具 - 工具名 - 场景` | `工具 - async-profiler - CPU 火焰图` |
| 问题追踪记录 | 持续追踪开放问题 | 问题尚未解决或需要验证 | `问题 - 主题` | `问题 - 场景服 P99 抖动来源` |

## 通用 Frontmatter

```yaml
---
tags: []
aliases: []
status: todo
created: {{date}}
updated: {{date}}
area: []
source: []
related: []
---
```

## 状态流转

- `todo`：刚创建，未加工。
- `draft`：已经整理，有初步结构。
- `review`：需要验证、补案例或补数据。
- `evergreen`：可长期复用。
- `archive`：过期或不再维护。
